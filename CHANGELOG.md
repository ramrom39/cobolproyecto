# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0] - 2026-03-15

### ✨ Agregado
- Interfaz gráfica Python (Tkinter) completa
- Pestaña de Cuentas Maestro con búsqueda en tiempo real
- Pestaña de Movimientos con validación de datos
- Pestaña de Procesamiento con log detallado
- Pestaña de Reportes con exportación a archivo
- Programa COBOL compilable (procesador_simple.cbl)
- Versión web en HTML5 con CSS3
- Procesador Python puro (alternativa sin Tkinter)

### 🎯 Características
- ✅ Carga de archivos maestro (formato ancho fijo)
- ✅ Procesamiento de movimientos (depósitos y retiros)
- ✅ Actualización de saldos en tiempo real
- ✅ Generación de reportes formateados
- ✅ Búsqueda y filtrado de cuentas
- ✅ Validación automática de datos
- ✅ Manejo de errores robusto

### 🔧 Mejoras
- Parseo correcto de archivos en ancho fijo
- Soporte para caracteres especiales (acentos, ñ, etc.)
- Interfaz responsive y moderna
- Código documentado
- Ejemplos de datos incluidos

### 📚 Documentación
- README.md completo
- Guía de contribución
- Licencia MIT
- .gitignore configurado
- requirements.txt
